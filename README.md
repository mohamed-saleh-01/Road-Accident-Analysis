# 🚗 Road Accident Analysis & Safety Insights

## 📌 Project Overview
This project provides an end-to-end data analysis solution to identify patterns, causes, and consequences of road accidents. Using a comprehensive dataset, it explores the impact of weather conditions, road types, and speed limits on accident severity.

The project combines rigorous data cleaning and Exploratory Data Analysis (EDA) in Python with an interactive Power BI dashboard to provide actionable safety insights.

---

## 📂 Dataset
- Source: Road Accident Dataset (Excel/CSV)
- Content: Information on accident severity, weather conditions, vehicle types, age groups, and casualties.

Key metrics analyzed:
- Total Accidents & Casualties
- Weather Impact (Rainy, Snowy, Foggy, etc.)
- Road Infrastructure (Width, Speed Limits)
- Demographic distribution of casualties.

## 🎯 Objectives
- Identify the most dangerous weather conditions for driving.
- Analyze the relationship between speed limits and accident frequency.
- Segment casualties by age group and vehicle type (Motorcycle, Car, Truck, etc.).
- Visualize the distribution of accidents relative to the nearest hospital distance.
- Provide a data-driven basis for road safety improvements.

---

## 🛠 Tech Stack
- Python: Pandas, NumPy, Matplotlib, Seaborn (Data Cleaning & EDA)
- Data Processing: Power Query
- Visualization: Power BI (DAX)
- Data Source: Microsoft Excel

---

## 🔄 Project Workflow
1. Data Cleaning: Handling missing values in visibility/weather columns and standardizing vehicle categories.
2. EDA (Python): Analyzing correlations between visibility and accident rates; performing time-series analysis.
3. Feature Engineering: Creating age group buckets (Adult, Senior, Teen, Young) and casualty ratios.
4. Dashboard Development: Designing an interactive UI with custom slicers for weather and demographic filters.

---

## 📊 Dashboard Insights

The Power BI dashboard reveals critical safety patterns:

- Weather Impact: Rainy and Snowy conditions show a significant spike in casualties, highlighting the need for better road traction and visibility measures.

- Vehicle Segmentation: Motorcycles and Pedestrians represent a high-risk group in urban areas, particularly at specific road widths.

- Infrastructure: Accidents are highly frequent in zones with an 81K speed limit profile, suggesting a need for stricter enforcement in these areas.

📌 Key Insight: Over 5K vehicles were involved in 1.61K recorded accidents, with visibility being a major contributing factor in nearly 891 cases.
